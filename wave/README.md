# Wave

#### Usage
```bash
cargo run -p wave -- [OPTIONS] 

OPTIONS:
    -s               Run in sequential model
    -p               Run in parallel mode
```

#### Examples:
- Runs the program in sequential mode
    ```bash
        $ cargo run -p wave -- -s
    ```
- Runs the program in parallel mode
    ```bash
        $ cargo run -p wave -- -p
    ```
